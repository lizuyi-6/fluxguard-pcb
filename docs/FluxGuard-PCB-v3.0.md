# FluxGuard Deployment Dongle - PCB v3.0 设计规范

**版本**: v3.0  
**日期**: 2026-03-15  
**设计理念**: 专业级、模块化、高可靠性

---

## 📋 设计目标

### 抛弃的问题
- 旧版设计过于简单，缺乏ESD保护
- GPIO分配不合理，存在冲突风险
- 电源管理缺失，依赖USB供电
- 没有硬件看门狗
- 缺乏调试接口

### v3.0 改进
1. **完整电源管理**：支持USB供电 + 锂电池备份
2. **ESD保护**：所有对外接口增加TVS管
3. **硬件看门狗**：独立监控ESP32状态
4. **Type-C全功能**：支持USB 2.0 + 调试
5. **模块化设计**：核心板 + 扩展板分离

## 📐 PCB 尺寸与布局

### 板卡尺寸
- **核心板**: 52mm x 29mm x 1.6mm (4层板)
- **扩展板**: 52mm x 29mm x 1.6mm (2层板)
- **间距**: 2.54mm排针连接

### 层叠结构 (4层板)
```
Layer 1 (Top):    信号层 + 电源
Layer 2 (GND):    完整地平面
Layer 3 (Power):  3.3V / 5V / VBATT
Layer 4 (Bottom): 信号层 + 电源
```

### 布局分区
```
┌─────────────────────────────────────┐
│  [USB-C]  [CH340]    [TP4056]       │ ← 电源/通信区
├─────────────────────────────────────┤
│  [ESP32]     [Flash]   [XTAL]       │ ← 核心区
│                                     │
│  [ATECC608]  [MAX6818]              │ ← 安全区
├─────────────────────────────────────┤
│  [OLED FPC]  [WS2812]  [BUZZER]     │ ← 显示/交互区
│                                     │
│  [BTN OK]  [BTN NEXT]  [BTN BACK]   │
├─────────────────────────────────────┤
│  [UART HEADER]  [DEBUG HEADER]      │ ← 扩展接口区
└─────────────────────────────────────┘
```

## 🔒 安全设计

### ESD保护
- USB接口：SRV05-4 (±30kV)
- 按键接口：100Ω串联电阻 + TVS
- UART接口：SRV05-4

### 加密存储
- ATECC608A存储：
  - 设备唯一ID
  - 私钥 (ECC P-256)
  - 证书链
  - 一次性可编程配置

### 硬件看门狗
- MAX6818 超时：1.6秒
- 喂狗周期：1秒
- 超时动作：复位ESP32

## 📦 接口定义

### USB-C 引脚
| Pin | 信号 | 连接 |
|-----|------|------|
| A1/A12/B12 | GND | 地 |
| A4/A9/B4/B9 | VBUS | 5V |
| A6/B6 | D+ | CH340 D+ |
| A7/B7 | D- | CH340 D- |
| A5/B5 | CC1/CC2 | 5.1K下拉 |

### 目标服务器UART (2x5pin 2.54mm)
| Pin | 信号 | 方向 |
|-----|------|------|
| 1 | VCC | 输出 |
| 2 | GND | - |
| 3 | TX | 输出 → 服务器RX |
| 4 | RX | 输入 ← 服务器TX |
| 5 | RTS | 输出 |
| 6 | CTS | 输入 |
| 7-10 | NC | - |

### 调试接口 (2x3pin 1.27mm)
| Pin | 信号 |
|-----|------|
| 1 | VCC |
| 2 | GND |
| 3 | TX |
| 4 | RX |
| 5 | EN |
| 6 | IO0 |

## 🏭 生产要求

- 层数：4层
- 板厚：1.6mm
- 铜厚：1oz (外层) / 0.5oz (内层)
- 表面处理：ENIG (金手指)
- 阻焊：黑色
- 丝印：白色
- 最小线宽/间距：6/6mil
- 最小过孔：0.3mm/0.5mm

### 测试点
- 电源测试点：3.3V, 5V, VBATT
- 信号测试点：TX, RX, SDA, SCL
- 编程测试点：EN, IO0, TX, RX

### 认证要求
- CE/FCC认证
- RoHS合规
- ESD测试：±8kV接触放电
